Elementos para trabajar una red

Internet: red de computadoras que se encuentran interconectadas a nivel mundial para compartir información.

Intranet: red informática interna de una empresa u organismo, basada en los estándares de internet, en donde las computadoras están conectadas a uno o varios servidores web.

Extranet: es una red privada que utiliza protocolos que se extienden más allá de los límites físicos de una corporación. Da acceso a vendedores, proveedores, distribuidores a la intranet de una compañía.

Puente de red: es un dispositivo de interconexión de ordenadores que opera en la capa 2 del modelo OSI, éste interconecta dos segmentos de red haciendo el pasaje de datos de una red hacia otra; con el destino de cada paquete.

Red: conjunto de computadoras, otros equipos interconectados, que comparten información, recursos y servicios según el alcance.

Protocolos: conjunto de reglas predefinidas con el propósito de estandarizar el intercambio de información en actividades informáticas.

TCP/IP: modelo que es un protocolo para comunicación en redes que permite que un equipo pueda comunicarse dentro de una red.

IPv4, IPv6: son estándares de direcciones IP que definen como se asigna una dirección IP y cuál será.

Direcciones físicas: es la ubicación de una unidad de memoria.

Direcciones lógicas: es la dirección virtual generada por la CPU.

Direccionamiento: son diferentes maneras de especificar un operando dentro de una instrucción en lenguaje ensamblador.

IP: es una representación que identifica una interfaz concreta de manera única en la red.

Modelo de comunicación guiado: aquellos que se utilizan en componentes físicos y sólidos para la transmisión de datos.

Modelos de comunicación no guiado: medio para cubrir grandes distancias y hacia cualquier dirección para transmitir datos.
