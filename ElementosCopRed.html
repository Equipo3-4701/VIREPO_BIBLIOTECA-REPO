Elementos que componene una red

Módem: dispositivo que convierte señales digitales en analógicas o viceversa.

Servidor: es un tipo de software que forma parte de una red informática y puede determinar servicios al resto de computadoras de la misma, llamadas a su vez clientes.

Switch: dispositivo analógico que permite interconectar redes operando en la capa 2 o de nivel de enlace de datos del modelo OSI.

Hub: es un dispositivo que canaliza el cableado de una red para ampliarla y repetir la misma señal a través de diferentes puertos.

Router: dispositivo de hardware que permite la interconexión de ordenadores en red.

Repetidor: dispositivo electrónico que recibe una señal débil o de bajo nivel y la retransmite a una potencia o nivel más alto.

Punto de acceso: dispositivo de red que interconecta equipos de comunicación inalámbrica para formar una red.

Balanceador: dispositivo que actúa distribuyendo el tráfico de red o de una aplicación a varios servidores.

Par trenzado: forma de conexión en la que dos aisladores son trenzados para tener menores interferencias y aumentar la potencia de cables adyacentes.

Conectores: son interfaces para conectar dispositivos mediante cables.

Patch panel: sirve como organizador de conexiones de la red, para que los elementos relacionados de la red LAN y los equipos de la conectividad puedan ser fácilmente incorporados al sistema.

Patch cored: conecta un dispositivo electrónico con otro.
