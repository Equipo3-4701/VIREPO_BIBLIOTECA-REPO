Practica 1: Reconocimiento

Se nos mostrara el funcionamiento del software Arduino en conjunto al simulador Proteus, esto con la finalidad de adaptarnos a las respectivas interfaces de cada programa.

Practica 2: Semaforo

Se nos ha pedido programar el funcionamiento de un semáforo utilizando el software Arduino y Proteus, el cual ejecutara las acciones correspondientes: led verde, posteriormente led amarillo y finalizando con el led rojo.

Practica 3: Sensor de Temperatura

Se nos ha pedido programar el funcionamiento de un sensor de temperatura utilizando el software Arduino y Proteus, este mostrara en la pantalla LCD la temperatura que ha sido asignada por el sensor.

Practica 4: Motores

Se nos ha pedido programar el funcionamiento de un motor de rotación utilizando el software Arduino y Proteus, este tendrá que rotar de acorde a las manecillas del rejo por 5 segundos.

Practica 5: Actuador Electrico

Se nos ha solicitado generar un ensamble de manera virtual en el cual debe ser visible el proceso donde giran dos motores en sentido contrario uno del otro implementando un actuador eléctrico.

Practica 6: Potenciometro

Se nos ha solicitado generar un ensamble de manera virtual en el cual debe ser visible el proceso donde giran dos motores en sentido contrario uno del otro implementando un actuador eléctrico.

Practica 7: Barra Transportadora

Se nos ha solicitado diseñar una banda transportadora que realice el traslado de muestras de un laboratorio a otro, hay que tener en cuenta dos motores que giran en ambos sentidos izquierda y derecha.

Practica 8: Maquina Centrifugadora

Se nos ha solicitado diseñar una maquina centrifuga que incluya una perilla para ajustar el tiempo de funcionamiento del rotor, el tiempo mínimo es de 10 segundos y el tiempo máximo es de 1 minuto.
