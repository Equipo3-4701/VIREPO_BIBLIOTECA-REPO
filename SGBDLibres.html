Sistemas Gestores de BD Libres

MYSQL

  • Características:

    • Facilidad de uso y gran rendimiento Facilidad para instalar y configurar Soporte multiplataforma Soporte SSL.

  • Requerimientos:
  • S.O: Windows, Linux y Unix 32/64bits.
  • Memoria RAM: 512MB.
  • Memoria: 1GB.
  • Versiones:
  • MYSQL 2.5.
  • MYSQL 2.9.1.
  • Distribuidor:
  • GNU/GPL.
  • Costo: Ninguno.

MariaDB

  • Características:

    • Aumento de motores de almacenamiento Gran escalabilidad Seguridad y rapidez en transacciones Extensiones y nuevas características relacionadas con su aplicación para Bases de datos NoSQL.

  • Requerimientos:
  • S.O: 32 ó 64 bits.
  • Memoria RAM: 512 MB.
  • Memoria: 1GB.
  • Versiones:
  • MYSQL 5.1.
  • MYSQL 5.5.
  • MariaDB 5.1.
  • MariaDB 5.2.
  • MariaDB 5.3.
  • MariaDB 5.5.
  • Distribuidor:
  • Linux
  • Costo: Ninguno.

SQLite

  • Características:

    • El tamaño, al tratarse de una biblioteca, es mucho menor que cualquier SGBD Reúne los cuatro criterios ACID (Atomicidad, Consistencia, Aislamiento y Durabilidad) logrando gran estabilidad Gran portabilidad y rendimiento

  • Requerimientos:
  • S.O: Windows xp/2003/vista/7/9x, Linux y MacOS.
  • Memoria RAM: 2GB.
  • Memoria: 9GB.
  • Versiones:
  • SQLite 1.4.2.
  • Distribuidor:
  • Navicat.
  • Costo: Ninguno.

PostgresSQL

  • Características:

    • Control de Concurrencias multiversión (MVCC) Flexibilidad en cuanto a lenguajes de programación Multiplataforma Robustez, Eficiencia y Estabilidad.

  • Requerimientos:
  • S.O: Windows, Linux y Unix.
  • Memoria RAM: 512MB.
  • Memoria: 1 GB.
  • Versiones:
  • PostgresSQL 6.3 hasta 9.2 (actual).
  • Distribuidor:
  • Postgres.
  • Costo: Ninguno.

Cassandra

  • Características:

    • Multiplataforma Propio lenguaje de consultas (CQL) Escalado lineal y horizontal Es un SGBD distribuido Utiliza una arquitectura peer-to-peer.

  • Requerimientos:
  • S.O: Java 8 y Python 2.7.
  • Memoria RAM: 2GB.
  • Memoria: 2GB.
  • Versiones:
  • Unica version.
  • Distribuidor:
  • Apache.
  • Costo: Ninguno.
